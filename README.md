# Germany CO₂ Emissions Analysis

## Project Overview
- Analyzed Germany's national CO₂ emissions (1970-2022)
- Adapted to focus on climate change
- Demonstrates data pipeline methodology for societal threat monitoring

## Key Findings
- **Trend:** -9.26 Mt CO₂/year (1970-2022)
- **R²:** 0.93 (excellent fit)
- **Reduction:** 42% since 1990 peak
- If sustained: Will reach 2030 climate targets by 2045

## Files
- `main.go`: Entry point
- `process.go`: Data loading/cleaning
- `model.go`: Trend analysis
- `save.go`: Processed dataset generation
- `prompts.md`: AI validation log
